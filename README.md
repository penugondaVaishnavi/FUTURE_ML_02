# Support Ticket Classification System

This project is part of the Machine Learning Internship at Future Interns.

## 📌 Project Overview
The goal of this project is to automatically classify customer support tickets into different categories using Machine Learning.  
It also assigns priority levels to tickets for efficient handling.

## 📊 Dataset
- A sample dataset of support tickets is created.
- Features:
  - Text: Description of the issue
  - Category: Type of issue (Billing, Technical, Security, etc.)

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- CountVectorizer (for text processing)

## 🤖 Model Used
- Naive Bayes (MultinomialNB) is used for text classification.

## 📈 Project Workflow
1. Data Creation  
2. Text Preprocessing  
3. Feature Extraction (CountVectorizer)  
4. Model Training  
5. Ticket Classification  
6. Priority Assignment  

## 🔮 Output
- The model predicts the category of new support tickets.
- A rule-based system assigns priority (High / Medium / Low).

## 🧠 Skills Gained
- Text Preprocessing  
- Natural Language Processing (NLP)  
- Machine Learning Classification  
- Feature Extraction  
- Problem Solving  

## 📌 Conclusion
This project demonstrates how Machine Learning can automate support ticket classification and improve response efficiency by assigning priorities.
